# Simple Mail Server

A simple Node.js package to send emails using **Nodemailer**.  
Designed for developers to quickly send emails using **built-in templates** or **custom HTML** with minimal setup.

---

## Features

- Send emails with a single function call
- Supports **custom HTML content**
- Supports **built-in templates** (like welcome emails)
- Debug option for logging email info
- Returns structured success/error objects

---

## Installation

```bash
npm install simple-mail-server
